# 🌆 Horizon Boards Toronto

A modern, responsive **billboard advertising landing page** designed for a fictional outdoor advertising agency based in Toronto.  
This project focuses on clean UI design, brand positioning, and digital representation of real-world out-of-home (OOH) advertising services.

---

## Overview

**Horizon Boards Toronto** is a conceptual advertising agency website showcasing billboard marketing services across high-traffic urban locations.

The goal of this project is to simulate a real-world marketing landing page that communicates:
- Brand visibility through outdoor advertising
- High-impact visual messaging
- Urban advertising placements across Toronto

---

## Features

- Clean and modern landing page design
- Billboard advertising service sections
- Fully responsive layout (mobile + desktop friendly)
- Marketing-focused UI/UX structure
- Easy navigation and clear content hierarchy

---

## Tech Stack
- HTML5
- CSS3
- Responsive Design (Flexbox / Media Queries)

---

## Purpose of the Project

This project was built to practice:
- Frontend layout design using pure HTML & CSS
- Structuring a marketing/agency-style website
- Building visually appealing UI without frameworks
- Understanding real-world advertising website layouts

---

## Use Case Simulation

This website simulates a billboard advertising agency that helps businesses:
- Promote brands across Toronto’s busiest streets
- Increase visibility through outdoor media placements
- Run high-impact advertising campaigns on digital & static billboards
